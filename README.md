Project Name

Game Zone - Luck Your Chance & Tic Tac Toe

Description

This project is a simple gaming application that contains two fun games:

Luck Your Chance
A luck-based game where users can try their fortune and get random results.
Tic Tac Toe
A classic 2-player Tic Tac Toe game with an interactive game board.

The project is created for learning and entertainment purposes.

Features
Two games in one project
Simple and user-friendly interface
Responsive design
Interactive gameplay
Easy to run and modify
